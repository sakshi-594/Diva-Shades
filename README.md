## Features
- Responsive UI with dark rose-gold theme
- Product listing, search & filter
- Dynamic Add to Cart & LocalStorage
- Cart quantity management, tax calculation
- Checkout page with validation

## Tech Stack
HTML5 • CSS3 • Bootstrap • JavaScript • LocalStorage

## How to run locally
1. Clone the repo: `git clone https://github.com/<sakshi-594/diva-shades.git`
2. Open `index.html` in a browser or serve with a simple server:
   - `npx http-server` or `python -m http.server 5500`
3. Enjoy!
